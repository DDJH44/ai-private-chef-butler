**总计**：约 1 小时

## P1 — 饮食拍照分析（热量/营养识别）

**目标**：拍饭菜照片 → AI 识别食物 + 估算热量/蛋白质/碳水/脂肪，不推荐菜谱

| 步骤 | 内容 | 工时 |
|------|------|------|
| 2.1 | System Prompt 新增「食物分析模式」：识别每道食物，估算分量和热量 | 0.5h |
| 2.2 | ChatInput 新增 📊 模式切换按钮（推荐菜谱 / 分析食物） | 1h |
| 2.3 | 分析结果结构化输出：食物名 / 重量 / 热量 / 蛋白质 / 碳水 / 脂肪 | 0.5h |
| 2.4 | 前端营养卡片展示：表格 + 进度条可视化 | 1.5h |
| 2.5 | 分析结果自动保存到今日饮食记录（联动功能三） | 0.5h |

**总计**：约 4 小时
